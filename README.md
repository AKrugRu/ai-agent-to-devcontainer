# Dev Container Features: Тестовый набор

Тестовый набор для дополнения devcontainer'ов утилитами cli AI агентов

Тест работоспособности:

```
# Запуск в указанном image
devcontainer features test --features <имя агента> --base-image mcr.microsoft.com/vscode/devcontainers/base:ubuntu

# Запуск сценариев
devcontainer features test --skip-autogenerated .
```
